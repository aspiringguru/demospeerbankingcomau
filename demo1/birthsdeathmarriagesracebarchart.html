Death by Disposal type 2010-2019

Data from data.qld.gov.au

This page uses data collated from data.qld.gov.au We used python pandas.read_excel to load spreadsheet files, reformat data with numpy & pandas, then exported to json. Race Bar chart is created using canvas, animations and rectangles drawn dynamically with interpolation of data between the data points given to create a continuous flow. Labels and numbers are dynamically positioned to ensure readability. Vertical position of horizontal rectangles is adjusted to create the race bar chart visual effect. Rectangles can be replaced with images/icons as required.
